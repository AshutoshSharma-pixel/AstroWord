# AstroWord SEO Audit File Bundle

This folder contains the files requested for an SEO/indexing review.

Note: this repo does not contain `.mdx` blog posts. Blog articles are defined in `frontend/src/app/blog/[slug]/page.tsx`, so that file is included as the existing blog post format reference.

Included:
- `frontend/src/app/layout.tsx`
- `frontend/src/app/sitemap.ts`
- `frontend/src/app/blog/[slug]/page.tsx`
- All 7 calculator route `page.tsx` files
- All 7 calculator route `layout.tsx` files
